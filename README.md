## 🚀 Getting Started

**Prerequisites:**

- Install Python 3.10 or later.
- Create a virtual environment:
  ```bash
  conda create --name Scraping_env python=3.10
  conda activate Scraping_env  
- Install requirement :
```bash
  pip install -r requirements.txt
```
